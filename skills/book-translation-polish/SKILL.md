# Book Translation Polish

## Core Rules

Use this skill when revising a Chinese Markdown draft against the English source of a book chapter, preface, or afterword.

1. Completely revise the Chinese draft. Freely adjust syntax, rewrite, paraphrase, and naturalize idioms so the result reads as if originally written by a native Chinese writer. Prefer fluent Chinese academic prose over English-shaped sentence order.

2. Do not omit content. Paraphrase is allowed, but every substantive claim, example, qualification, contrast, date, number, and cited source in the English original must remain represented in Chinese.

3. Renumber notes per chapter or standalone file starting from 1. Adjust all in-text note references and final note entries consistently.

4. Translate explanatory note content into Chinese. In final notes, keep bibliographic metadata such as book titles, article titles, journal names, citation author names, publisher names, publication places, dates, page ranges, and URLs in English. For people mentioned in prose outside citation metadata, apply the verified-name rule. Do not leave explanatory prose in English merely because it appears in a note.

5. For quotations in the English source that originate from Mao Zedong, Chinese newspapers, party documents, or other Chinese-language sources, search for the Chinese original when feasible. If a reliable Chinese original is found, replace the translated-back English quotation with the Chinese original. If no reliable Chinese original is found, translate it carefully and mark it with `【原文待核】` near the quotation or in a translator note so a human can revisit it.

6. For personal names, especially scholars and public intellectuals, use a commonly accepted Chinese name only after verifying it through reliable web sources such as Chinese Wikipedia, Baidu Baike, academic publisher pages, library authority records, or widely used scholarly references. If no accepted Chinese name is found, keep the English name. Avoid inventing transliterations.

## Workflow

1. Extract or view the English source for the target file and identify its exact boundaries.

2. Compare the English source to the Chinese draft for omissions, compression, mistranslation, awkward literalism, and note mismatch.

3. Search the web when needed to verify accepted Chinese personal names and to locate original Chinese quotations. Record uncertainty inline with `【原文待核】` only when the original cannot be reliably found.

4. Rewrite the file as polished Chinese, preserving Markdown structure unless the source requires adding missing sections.

5. Rebuild note numbering from 1 for the file. Ensure every in-text note reference has a corresponding note entry and every note entry is cited in the text, except when a deliberate uncited bibliographic note is explicitly justified.

6. Run a mechanical check after editing: count note references, note entries, gaps, and uncited notes; inspect changed passages manually where the check reports problems.

## Style Guidance

Use natural Chinese paragraph rhythm. Break up long English periods where Chinese needs shorter clauses. Prefer established Chinese historical and political terminology when appropriate, but do not flatten the author's argument. Keep dates, quantities, proper nouns, and argumentative nuance precise.

When replacing an English idiom or colloquial phrase, choose an idiomatic Chinese expression, chengyu, allusion, or plain native phrasing that preserves tone and meaning. Do not over-ornament academic prose.
